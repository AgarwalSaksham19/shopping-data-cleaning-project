# Shopping Dataset Data Cleaning Project

## Objective
To perform data cleaning and basic analysis using Python and Pandas.

## Steps Performed
- Loaded dataset into Pandas DataFrame
- Selected relevant columns
- Removed duplicates
- Handled missing values
- Converted and cleaned numeric fields
- Performed filtering operations
- Created derived feature (value_score)
- Saved cleaned dataset

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Output
- Cleaned dataset CSV file
- Data analysis notebook
